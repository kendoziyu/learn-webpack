# learn-webpack
learn to config webpack from scratch

#  准备工作：
下载代码到本地后，需要执行安装
+ `npm install`
+ `yarn install`

|demoX|描述|
|---|---|
|demo1|Webpack 基础配置|
|demo2|HtmlWebpackPlugin 配置|
|demo3|css 文件打包|
|demo4|把 css 样式抽取到文件及压缩|
|demo5|使用babel-loader 支持 es6+ 语法|
|demo6|使用 plugin-transform-runtime 及 eslint |
|demo7|引入全局变量|
|demo8|引用图片|
|demo9|打包多页应用|
